SECTION 5 — KEY STATISTICAL FINDINGS (ready to cite in paper)
────────────────────────────────────────────────────────────────────────

Use these statements directly in your Results section:

  [Executability — GPT-120B]
  All strategies achieved identical executability (100%) on GPT-OSS-120B,
  producing no variance and rendering statistical comparison impossible.
  This is itself a meaningful finding: large models are prompt-agnostic
  with respect to syntactic correctness.

  [Executability — LLaMA-70B]
  Check Section 2 Friedman results above. If p < 0.05, cite:
  'Prompting strategy had a statistically significant effect on executability
  for LLaMA-3.3-70B (Friedman χ²=X, p=Y). Pairwise analysis (Wilcoxon,
  Bonferroni-corrected) showed that CoT significantly outperformed Zero-Shot
  (p=Z, Cliff's δ=W, large effect).'

  [Mutation Score dissociation]
  'Despite near-zero pass rates, LLaMA-3.3-70B CoT achieved 80% mutation
  score, demonstrating that mutation score and pass rate can dissociate.
  This was not observed in GPT-OSS-120B, where both metrics were correlated.
  Practitioners should not rely on mutation score alone as a proxy for
  test correctness.'

  [Partition List]
  'partition_list was the only function where all models × strategies
  failed to kill the mutant, identifying linked list manipulation as a
  systematic failure mode for current LLM test generation.'
